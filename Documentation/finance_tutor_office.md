# EO-083 Finance Tutor Office

## Mission

The Finance Tutor Office serves as each student's personal financial mentor by drawing on validated ARGOS institutional intelligence, doctrine, historical examples, case studies, and supporting evidence.

## Implemented Capabilities

- Personalized Tutoring Framework
- Adaptive Guidance Architecture
- Socratic Reasoning Framework
- Explanation Depth Standard
- Decision Coaching Framework
- Misconception Detection Framework
- Educational Conversation Model
- Tutor Analytics Dashboard
- Finance Tutor System Prompt

## Deterministic Tutoring

Tutoring sessions are generated from learner context, demonstrated competencies, misconceptions, educational goals, learning history, and traceable knowledge references. Guidance mode and explanation depth are selected by deterministic rules.

## Educational Boundary

The Finance Tutor Office provides evidence-based mentoring and decision coaching. It does not provide investment advice, alter doctrine, create institutional knowledge, or bypass the Academy's traceability requirements.
