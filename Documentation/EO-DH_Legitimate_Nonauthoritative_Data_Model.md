# EO-DH Legitimate Nonauthoritative Data Model

Generated data may remain only when isolated to test, proof, simulation, replay, Decision Laboratory, or display-only domains.

EO-DH defines quarantine namespaces for TEST, PROOF, SIMULATION, REPLAY, and DISPLAY artifacts. These namespaces prohibit promotion into Trader, Broker, Position Registry, Performance Truth, Closed Position Truth, Historian, learning, and persistence sinks.
