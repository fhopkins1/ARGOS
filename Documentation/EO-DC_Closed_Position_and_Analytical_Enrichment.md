# EO-DC Closed Position and Analytical Enrichment

Closed Position Truth facts must come from authoritative Broker fills, position state, reconciliation, and accounting.

Analytical enrichment such as benchmark comparison, attribution, counterfactual analysis, strategy classification, and learning features is separate from the underlying fact.

Existing Phase III.5 behavior keeps degraded enrichment outside authoritative Closed Position Truth by default. EO-DC reinforces that degraded enrichment cannot become an authoritative financial fact or certified learning input automatically.
