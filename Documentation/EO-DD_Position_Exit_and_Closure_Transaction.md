# EO-DD Position Exit and Closure Transaction

Full-closure transactions require Paper Broker, Position Registry, Performance Truth, Closed Position Truth, and Historian participation.

EO-DD requires each authority to acknowledge durable evidence before a closure can be considered committed. Duplicate closed-position truth is treated as a blocking reconciliation discrepancy through the EO-DA broker-position invariant monitor.

EO-DD cannot create closed-position truth and cannot mark a position closed without participant evidence.
