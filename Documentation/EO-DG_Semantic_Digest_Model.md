# EO-DG Semantic Digest Model

EO-DG provides versioned SHA-256 digest profiles:
- `DIGEST-RUNTIME`
- `DIGEST-LAWVII`
- `DIGEST-BROKER`
- `DIGEST-POSITION`
- `DIGEST-TRUTH`
- `DIGEST-HISTORIAN`
- `DIGEST-TRANSACTION`
- `DIGEST-GOVERNANCE`
- `DIGEST-COST`
- `DIGEST-ALL-AUTHORITATIVE`

Normalization sorts keys, normalizes enum values, rounds floats, and excludes permitted ephemeral metadata. Profile-specific projection keeps unrelated domains separate.
