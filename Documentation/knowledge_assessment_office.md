# EO-081 Knowledge Assessment Office

## Mission

The Knowledge Assessment Office evaluates each student's financial reasoning, investment judgment, competency development, and application of validated institutional knowledge.

## Implemented Capabilities

- Competency Assessment Framework
- Adaptive Assessment Architecture
- Student Competency Profile Standard
- Decision-Based Evaluation Framework
- Educational Feedback Framework
- Longitudinal Learning Analytics
- Assessment Validation Process
- Assessment Dashboard
- Knowledge Assessment Prompt

## Deterministic Assessment

Assessments measure evidence evaluation, decision quality, risk recognition, critical thinking, and institutional knowledge application. Competency scores are calculated from recorded assessment responses and required evidence coverage using deterministic procedures.

## Adaptive Testing

Future assessment items are selected from measured knowledge gaps. When no gaps are detected, the architecture advances toward more difficult assessment items.

## Boundary

The Knowledge Assessment Office measures educational competency. It does not provide investment advice, create new institutional knowledge, or replace Curriculum Office pathway governance.
