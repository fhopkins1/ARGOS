# EO-065 Prompt Evaluation Office

## Mission

The Prompt Evaluation Office scientifically evaluates every prompt used throughout ARGOS. It treats prompts as organizational assets and operational hypotheses that must be measured empirically rather than accepted by opinion.

## Implemented Capabilities

- Prompt Performance Evaluation
- Prompt Experimentation
- Prompt Version History
- Prompt Benchmarking
- Prompt Quality Metrics
- Prompt Validation
- Prompt Optimization Recommendations
- Prompt Performance Reports
- Enterprise Prompt Analytics
- Prompt Evidence Repository

## Deterministic Controls

Prompt quality is measured from accuracy, consistency, completeness, reasoning quality, evidence usage, decision quality, latency, determinism, reproducibility, human evaluation, and organizational impact. Production validation requires determinism, historical improvement, evidence usage, organizational alignment, safety, reproducibility, statistical significance, and Executive approval.

## Governance

The office produces evidence-based recommendations but never directly modifies prompts. No prompt may progress toward doctrine without empirical validation and preserved performance evidence.

## Failure Detection

The office detects prompt regression, performance degradation, inconsistent outputs, unsupported conclusions, prompt drift, safety violations, reasoning failures, and weak evidence usage.
