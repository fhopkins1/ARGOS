# EO-DC Scope-Specific Certification

Certification scopes are defined by `PromotionScope` and `PROMOTION_SCOPE_REGISTRY`.

Current scopes include Analyst consideration, Risk evaluation, Trader order construction, Broker submission, Position Registry mutation, Performance Truth ingestion, Historian preservation, certified learning, Commander display, replay only, and simulation research.

A record valid for display is not automatically valid for mutation. A replay record is not automatically valid for certified learning or active paper truth.
