# EO-079 Instruction Office

## Mission

The Instruction Office translates validated ARGOS institutional intelligence into personalized, evidence-based financial instruction. It teaches how ARGOS reasons, not merely what ARGOS concludes.

## Implemented Capabilities

- Instruction Architecture
- Lesson Design Standard
- Educational Translation Framework
- Historical Learning Framework
- Decision Reconstruction Standard
- Personalized Instruction Framework
- Lesson Validation Process
- Instruction Dashboard
- Academy Instruction Prompt

## Deterministic Instruction

Lessons are generated from validated source materials that include institutional knowledge, doctrine, case files, research references, evidence, and historical outcomes. Lesson validity requires traceability, doctrine support, historical context, reconstruction evidence, and absence of individualized investment instructions.

## Personalization

Instruction adapts pacing and sequence by learner level. Novice and developing learners receive guided progression, while advanced learners receive source review, argument mapping, reconstruction, alternatives, and independent application.

## Boundary

The Instruction Office teaches financial reasoning and evidence discipline. It does not recommend trades, forecast markets for the user, or personalize investment instructions.
