# EO-DF Broker and Position Drift Model

EO-DF monitors orders, working orders, rejected orders, cancelled orders, expired orders, fills, partial fills, duplicate-event rejections, settlements, open positions, closed positions, position events, EO-CK enrollments, exit workflows, pending exits, and reconciliation discrepancies.

Drift is visible and blocking when unsafe. EO-DF does not normalize drift by overwriting one authority with another.
