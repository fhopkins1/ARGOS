# EO-DE Resource Boundedness

EO-DE records deterministic resource snapshots for every injected fault:
- memory units
- CPU units
- thread count
- task count
- queue depth
- cache entries
- EO-CK enrollments
- Scheduler obligations
- message count
- persistence records

Resource-exhaustion campaigns use bounded pressure and must remain below the configured queue and thread limits. Unbounded growth prevents PASS.
