# EO-025 Event Intelligence Office

The Event Intelligence Office is a deterministic Seeker office. It processes
supplied objective market-event observations and produces descriptive signals
for opportunities, threats, information gaps, and event assessments.

## Seekers

- Earnings Event Seeker
- Merger & Acquisition Seeker
- Bankruptcy Seeker
- Regulatory Event Seeker
- FDA Event Seeker
- Litigation Seeker
- Product Launch Seeker
- Geopolitical Event Seeker
- Supply Chain Event Seeker

## Reports

- Candidate Opportunity Report (`COR`)
- Event Intelligence Threat Report (`ETR`)
- Information Gap Report (`IGR`)
- Event Assessment Report (`EAR`)

## Boundaries

The office does not recommend trades, determine position sizing, execute trades,
bypass deterministic interfaces, or ignore Risk Office doctrine.
