# EO-DC Performance and Overhead Report

EO-DC is deterministic and local.

It does not call external models, wake analytical offices, perform runtime repository-wide scans, create unbounded promotion records, duplicate financial truth, or mutate financial state.

Focused EO-DC targeted tests ran 10 tests in approximately 0.01 seconds locally. The focused adjacent regression sweep ran 39 tests in approximately 0.58 seconds.
